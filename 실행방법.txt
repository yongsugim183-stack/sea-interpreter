동남아시아 + 중앙아시아 동시통역기 실행 방법
API 키 없이 완전 무료 사용 가능
================================================================

【무료 동작 원리】
  번역: Google Translate (deep-translator 라이브러리, API 키 불필요)
  STT:  faster-whisper 로컬 모델 (무료) 또는 브라우저 Web Speech API


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【방법 A】 서버 실행 (완전 무료, API 키 불필요)  ★ 권장
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. PowerShell 열기

2. 패키지 설치:
   cd C:\Users\김용수\sea-interpreter
   pip install -r requirements.txt

   ※ 첫 실행 시 Whisper 모델 자동 다운로드 (~244MB)
   ※ 이후 실행은 즉시 시작 (오프라인 가능)

3. 서버 실행 (API 키 없이):
   python server.py

4. 브라우저에서 접속:
   http://localhost:8000

5. 마이크 버튼 클릭 → 발화 → 자동 인식 & 9개 언어 동시 번역


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【STT 자동 선택 우선순위】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1순위: faster-whisper 로컬 (설치 시, 무료, 혼합언어 지원)
2순위: OpenAI Whisper API (OPENAI_API_KEY 설정 시, 유료)
3순위: 브라우저 Web Speech API (1,2 모두 없을 때, 무료, 단일언어)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【Whisper 모델 크기 선택 (선택사항)】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
기본값 = small (~244MB, 9개 언어 양호)

더 빠르게 (정확도 낮음):
  $env:WHISPER_MODEL = "base"    # ~74MB
  python server.py

더 정확하게 (느림):
  $env:WHISPER_MODEL = "medium"  # ~769MB
  python server.py


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【방법 B】 HTML 파일 직접 실행 (서버 없음)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. index.html을 Chrome 브라우저로 열기
2. 텍스트 입력 또는 마이크 버튼 클릭
3. 번역은 MyMemory 무료 API, STT는 Web Speech API 사용
※ 서버 없이 실행하면 정확도가 낮을 수 있습니다


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【서버 상태 표시 (화면 하단)】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🟢 Whisper(로컬) + Google Translate  → 완전 무료 최상 모드
🟢 Whisper(OpenAI) + Google Translate → 유료 STT + 무료 번역
🟡 Web Speech API + Google Translate  → 무료 (단일언어 STT)
🟡 서버 미연결 · MyMemory 번역        → 완전 오프라인 폴백
